web tutorial

step 1: folders

create folders under your root folder:

  • css
  • images
  • js
    ⬅ ️css folder (contains all your .css files)
    ⬅ images folder (contains all your images)
    ⬅ js folder (contains all your .js files)

step 2: html

create an index.html file:

  • type "html:5" to set it up
  • create an h1 tag in body
  • commence coding
    ⬅ "index.html" file (loads first)

link a page:
<a href="http://www.link-here.com">click hERE</a>

link an image:
<img src="images/image.png alt="image">

step 3: css

there are 3 ways to do css:

  • inline
  • internal
  • external stylesheet

do inline style:
<p style="color: red;">red text</p>

internal style in the <head>:
<style>
    h1 {
        font-weight: 900;
        font-size: 10em;
    }
</style>

do external style via stylesheet in <head>:
<link rel="styelsheet" href="css/style.css">
