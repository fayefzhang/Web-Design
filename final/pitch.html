Web Design Final Project Pitch

- faye zhang

mission statement

A suitable portfolio website will be created that showcases Faye's skills in web design, programming, art, robotics and rock climbing. The main pages in the nav bar will be work, about, contact, store and resume. All skills should be shown in a visual way (with data) under the about tab and work should contain a grid that displays all of Faye's artworks. This website should have one main color palette and design and that is present through all the pages. Additionally, the page designs should be original and responsive to all page sizes. This website will be created as a introduction to Faye and used to advertise her work and resume. It should be complete, colorful, and aesthetically pleasing to the eye.

I would like to work on this website because I am the only one who is allowed to work on it as it is my portfolio website. I would like to move from my previous model to something more customized for myself now that I know more about html, css and js. Since I already own the domain for the site I choose, when this project is over, this website will be used in place of the previous design.
